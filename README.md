## Weather Prediction Dashboard

A modern, responsive web application that provides real-time weather information for any city worldwide. The project uses the OpenWeatherMap REST API to fetch and display live weather data, including temperature, humidity, wind speed, and general weather conditions.


## Features
Search weather by city name globally
Real-time weather data retrieval using REST API
Displays temperature, humidity, wind speed, and weather conditions
Dynamic UI updates based on API response
Error handling for invalid city names or failed requests
Responsive design for desktop and mobile devices
Clean and modern user interface
##  Technologies Used
HTML5
CSS3
JavaScript (ES6+)
Fetch API / Async-Await
OpenWeatherMap API
